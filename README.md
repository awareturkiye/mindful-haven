# MindfulHaven

A modern, accessible mental health and self-harm support website.  
Built with pure HTML, CSS, and vanilla JavaScript — no framework or build step required.

## Pages

| Page | URL |
|---|---|
| Homepage | `/index.html` |
| What is Mental Health | `/pages/what-is-mental-health.html` |
| Mental Health Problems | `/pages/mental-health-problems.html` |
| Self-Harm | `/pages/self-harm.html` |
| Get Help Now | `/pages/get-help.html` |
| Resources | `/pages/resources.html` |
| About | `/pages/about.html` |
| Contact | `/pages/contact.html` |

## Deployment

This site is deployed on [Vercel](https://vercel.com) as a static site.  
No build command or server is required.

## Local Development

Simply open `index.html` in your browser. No server needed.
